## ðŸš€ Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/Anshkumar111/LinkUp_Chat-app.git
cd LinkUp_Chat-app
```

### 2. Setup Backend

```bash
cd backend
npm install
# Create a .env file with your MongoDB, JWT, and Cloudinary configs
npm run dev
```

### 3. Setup Frontend

```bash
cd ../frontend
npm install
npm run dev
```

### 4. Open in Browser
